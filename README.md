# homebrew-tap

Homebrew tap for [OpenASR](https://github.com/QuintinShaw/openasr), a
local-first, fail-closed speech-to-text CLI (no cloud, no telemetry).

## Install

```sh
brew install quintinshaw/tap/openasr
```

## Supported platforms

- macOS (Apple Silicon / arm64)
- Linux (x86_64, arm64)

For other platforms, or to install without Homebrew, see the
[release page](https://github.com/QuintinShaw/openasr/releases) or the
one-line installer described in the main repo's README.

## Updating

The formula in `Formula/openasr.rb` is kept in sync with tagged releases of
[QuintinShaw/openasr](https://github.com/QuintinShaw/openasr) by that repo's
release workflow.
